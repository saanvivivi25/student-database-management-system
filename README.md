This project is a simple Student Database Management System developed using Python. It stores student details such as USN (University Seat Number), Name, and marks in three subjects using a dictionary data structure.

The system allows the user to search for a student by entering either the name or USN. Once the student is found, the program retrieves and displays their details, calculates the average marks, and assigns a grade based on the average performance.

The grading system is defined as follows:

90 and above → Grade A
80–89 → Grade B
70–79 → Grade C
60–69 → Grade D
Below 60 → Satisfied
Menu-Driven Operations

This program can be extended into a menu-driven system, where users can choose different operations such as:

Display All Student Records
Shows details of all students in the database.
Search Student (by Name or USN)
Allows the user to find a specific student and view their details.
Calculate Average and Grade
Computes the average marks and assigns a grade automatically.
Add New Student Record (optional extension)
Insert new student details into the database.
Exit
Terminates the program.
